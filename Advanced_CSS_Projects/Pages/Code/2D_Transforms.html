2D Transforms

1.The translate() Method

The translate() method moves an element from its current position:

This div element is moved 50 pixels to the right, and 80 pixels down from its current position.

2.The rotate() Method

The rotate() method rotates an element clockwise or counter-clockwise.

This div element is rotated clockwise 45 degrees.
This div element is rotated counter-clockwise with 60 degrees.

3.The scale() Method

The scale() method increases or decreases the size of an element.

This div element is two times of its original width, and two times of its original height.
This div element is decreased to be half of its original width and height.

4.The scaleX() Method

The scaleX() method increases or decreases the width of an element.

This div element is two times of its original width.

5.The scaleY() Method

The scaleY() method increases or decreases the height of an element.

This div element is two times of its original height.

6.The skewX() Methode

The skewX() method skews an element along the X-axis by the given angle.

This div element is skewed 10 degrees along the X-axis.

7.The skewY() Method

The skewY() method skews an element along the Y-axis by the given angle.

This div element is skewed 10 degrees along the Y-axis.

8.The skew() Method

The skew() method skews an element into a given angle.

This div element is skewed 20 degrees along the X-axis, and 10 degrees along the Y-axis.

9.The matrix() Method

The matrix() method combines all the 2D transform methods into one.

The parameters are as follow: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())
