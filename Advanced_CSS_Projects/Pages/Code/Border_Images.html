Border Images

1. The border-image Property

The CSS border-image property allows you to specify an image to be used instead of the normal border around an element.

(a) Here, the middle sections of the image are repeated to create the border.

border-image: url(Border.jpg) 35 round;

(b) Here, the middle sections of the image are stretched to create the border

border-image: url(Border.jpg) 20 stretch;

The border-image - Different Slice Values

Different slice values completely changes the look of the border

(a) border-image: url(Border.jpg) 100% round;

(b) border-image: url(Border.jpg) 20% round;

(cs) border-image: url(Border.jpg) 30% round;
