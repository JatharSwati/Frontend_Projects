Box Shadow

The CSS box-shadow property is used to apply one or more shadows to an element.

1. Specify a Horizontal and a Vertical Shadow -

Specify a horizontal and a vertical shadow. The default color of the shadow is the current text-color.

This is a div element with a box-shadow.

2. Specify a Color for the Shadow -

The color parameter defines the color of the shadow.

A div element with a gray box-shadow.

3. Add a Blur Effect to the Shadow -

The blur parameter defines the blur radius. The higher the number, the more blurred the shadow will be.

A div element with a 5px blurred, maroon box-shadow.

4. Set the Spread Radius of the Shadow -

The spread parameter defines the spread radius.

A positive value increases the size of the shadow, a negative value decreases the size of the shadow.

A div element with a blurred, lightblue box-shadow, with a spread radius of 15px.

5. Set the inset Parameter -

The inset parameter changes the shadow from an outer shadow (outset) to an inner shadow.

A div element with a blurred, darkblue, inset box-shadow
