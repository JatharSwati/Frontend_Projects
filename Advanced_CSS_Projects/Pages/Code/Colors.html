Colors

1.RGB Colors

A color can be specified by using a predefined color name

Orange with rgb(255, 165, 0)

Blue with rgb(0, 0, 255)

Green with rgb(0, 128, 0)

Violet with rgb(238, 130, 238)

Maroon with rgb(128, 0, 0)

2.RGBA Colors

RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity for a color.

An RGBA color value is specified with: rgba(red, green, blue, alpha).
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

Orange with rgba(255, 165, 0, 0.3)

Blue with rgba(0, 0, 255, 0.3)

Green with rgba(0, 128, 0, 0.3)

Violet with rgba(238, 130, 238, 0.3)

Maroon with rgba(128, 0, 0, 0.3)

3.HSL Colors

An HSL color value is specified with: hsl(hue, saturation, lightness).

a.Hue is a degree on the color wheel (from 0 to 360)

b.Saturation is a percentage value: 100% is the full color.

c.Lightness is also a percentage; 0% is dark (black) and 100% is white

SteelBlue with hsl(207, 44%, 49%)

Tomato with hsl(9, 100%, 64%)

PaleVioletRed with hsl(340, 60%, 65%)

Peru with hsl(30, 59%, 53%)

Crimson with hsl(348, 83%, 47%)

4.HSLA Colors

HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity for a color.

An HSLA color value is specified with: hsla(hue, saturation, lightness, alpha), where the alpha parameter defines the opacity.
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

SteelBlue with hsl(207, 44%, 49%, 0.2)

Tomato with hsla(9, 100%, 64%, 0.2)

PaleVioletRed with hsla(340, 60%, 65%, 0.2)

Peru with hsla(30, 59%, 53%, 0.2)

Crimson with hsla(348, 83%, 47%, 0.2)

5.Opacity

The CSS opacity property sets the opacity for the whole element (both background color and text will be opaque/transparent).

The opacity property value must be a number between 0.0 (fully transparent) and 1.0 (fully opaque).

Crimson with opacity: 0.2

Crimson with opacity: 0.4

Crimson with opacity: 0.6

Crimson with opacity: 0.8

Crimson with opacity: 1.0
